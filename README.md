# D-CAM

To use the D-CAM type the following commands in Termux:

`apt-get install python3`

`apt-get install git`

`git clone https://github.com/WHI73-D3VI1/D-CAM`

`pip3 install requests`

`pip3 install colorama`

`cd D-CAM`

`python3 D-cam.py`
